# tv-audience-forecasting

Zadání pro Data Science & ML Engineering v oblasti televizního trhu. Analýza a predikce sledovanosti (share) v cílové skupině 15–54 let. 

Tento repozitář obsahuje Jupyter notebooky pro explorativní analýzu dat (EDA), predikativní modely.

**Technologický stack:** Python.

---

> **Poznámka:** Veškeré detailní popisy, metodika a analýzy jsou uvedeny přímo u jednotlivých úloh (Task 1, Task 2, Task 3).
